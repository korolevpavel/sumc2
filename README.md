## SUMK — учебный проект для 1С:Предприятие.Элемент

**SUMK** — это учебная конфигурация, разработанная в рамках Цифровой кафедры СПбПУ для участия в конкурсе дипломных работ Цифровых кафедр 1С.

Проект представляет собой облачное приложение (веб‑кабинет) на платформе **1С:Предприятие.Элемент**, предназначенное для учета закупок и продаж, ведения номенклатуры товаров, работы с пользователями и административных операций.

### Основные возможности

- **Товары**
  - Справочник номенклатуры
  - Хранение основных реквизитов товара
- **Закупки**
  - Учет операций закупки
  - Связь с товарами и контрагентами
- **Продажи**
  - Учет операций продажи
  - Аналитика по реализованным товарам
- **Пользователи**
  - Справочник пользователей
  - Роли и права доступа
- **Администрирование**
  - Служебные и административные объекты
  - Настройки конфигурации
- **РеквизитыДляПеревода**
  - Хранение банковских и платежных реквизитов организации
- **API**
  - Объекты для интеграции с внешними системами

### Используемые технологии

- **1С:Предприятие.Элемент** — облачная low‑code платформа для разработки веб‑кабинетов, порталов и приложений ([описание платформы](https://1cmycloud.com/)).
- **1C:Enterprise Development Tools (EDT)** — среда разработки для проектов формата `Проект.xbsl` / `Проект.yaml`.

Проект основан на типовом шаблоне **`SUMC`** (поставщик: Колесников Станислав, версия: `1.0-91`) и адаптирован под учебные задачи.

### Структура репозитория

- **`Проект.xbsl`, `Проект.yaml`** — файлы описания проекта EDT и метаданных конфигурации.
- **`Товары`** — объекты, связанные с номенклатурой.
- **`Закупки`** — объекты подсистемы закупок.
- **`Продажи`** — объекты подсистемы продаж.
- **`Пользователи`** — пользователи и роли.
- **`Администрирование`** — административные объекты и настройки.
- **`РеквизитыДляПеревода`** — реквизиты организации для платежей.
- **`API`** — объекты для интеграции и внешних взаимодействий.

### Как открыть проект

1. Установите **1C:Enterprise Development Tools (EDT)**.
2. Клонируйте репозиторий или скачайте архив с проектом.
3. В EDT выберите пункт **«Открыть существующий проект»** и укажите файл `Проект.xbsl` или каталог проекта.
4. Выполните необходимые действия для публикации/развертывания конфигурации в среде **1С:Предприятие.Элемент** согласно документации платформы.
